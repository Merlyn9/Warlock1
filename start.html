Kanban

A kanban board is a japanese method of keeping on task and to track the progress of work.
It is very effective and is used by some of the worlds most successful people. If they use it,
maybe we can take a deeper look into the Kanban. It sure has helped me out along the way.


Virtual Box

Another alternative when you can't live without your Windows, is a VirtualBox.
A virtual Box or a vagrant box is a seperate computer inside a sandbox on your desktop. You can load this box with Ubuntu or Kali Linux and commence your jammin. It use your hardware space but is dynamically allocated if you so choose. So it uses what your not using when it gets up and running.

NodeJS

NodeJS is a JavaScript Runtime that lets you write code, then run that code.
You also can write applications for client-side servers and run them.
With nodemon you can run an app in a development environment while
updating the code and see immediate changes while viewing the app.
This JavaScript program will help the way you see and understand the back end.
